# SQL Basics Review

This is a beginner SQL review project I created after completing Kaggle's Intro to SQL course.

The goal of this project is to reinforce the core SQL concepts I learned and explain them in simple language.

## Topics Covered

- SELECT
- FROM
- WHERE
- DISTINCT
- COUNT
- GROUP BY
- HAVING
- ORDER BY
- AS
- WITH
- JOIN

## Purpose

I completed the Kaggle Intro to SQL course, but some of the later exercises were challenging. This project is my reinforcement step to practice the concepts more clearly using simple example tables and plain-English explanations.

## Status

Beginner review project. I am using this to strengthen my ability to write and explain SQL queries independently.
